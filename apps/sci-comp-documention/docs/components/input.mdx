# Input

`Input` 是基于 Ant Design Input 的通用封装组件，保留 `label`、`helperText` 与 `invalid` 等增强能力。

## 使用示例

```tsx
import { Input } from '@sci-comp/core'

export function Example() {
  return <Input label="名称" helperText="请输入组件名称" />
}
```
