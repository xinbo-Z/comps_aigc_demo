# Input

`Input` 是基于 Ant Design Input 的通用封装组件，在透传 antd 输入能力的同时，补充了 `label`、`helperText` 与 `invalid` 这类轻量展示与可访问性增强。

## 使用示例

```tsx
import { Input } from '@sci-comp/core'

export function Example() {
  return <Input label="名称" helperText="请输入组件名称" />
}
```
