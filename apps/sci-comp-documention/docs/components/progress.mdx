# Progress

`Progress` 是基于 Ant Design Progress 的通用封装组件，用于展示任务完成度、处理进度和阶段状态。

## 实时预览

<PreviewBlock
  title="基础进度条"
  description="保留 antd Progress 的核心能力，适合展示线性进度、状态反馈和完成率信息。"
>
  <ProgressDocDemo />
</PreviewBlock>

## 使用示例

```tsx
import { Progress } from '@sci-comp/core'

export function Example() {
  return <Progress percent={60} />
}
```
