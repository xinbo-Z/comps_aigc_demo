# Progress

`Progress` 是基于 Ant Design Progress 的通用封装组件，用于展示任务完成度、处理进度和阶段状态。

它适合承载“当前推进到哪里”的反馈信息，例如上传进度、审核流转、发布任务完成率和阶段性指标概览。在线性、环形和仪表盘三种常见形态之间切换时，可以继续沿用 antd 的核心能力，同时保持统一的组件库导出方式。

## 适用场景

- 展示任务、请求或批处理的完成百分比
- 在卡片、概览区中用 circle 或 dashboard 提供紧凑型进度反馈
- 配合 `status` 表达成功、进行中、异常等阶段状态

## 基础用法

```tsx
import { Progress } from '@sci-comp/core'

export function Example() {
  return <Progress percent={60} />
}
```

## 状态反馈示例

```tsx
import { Progress } from '@sci-comp/core'

export function StatusExample() {
  return (
    <>
      <Progress percent={80} status="success" />
      <Progress percent={45} status="active" />
    </>
  )
}
```

## 选型建议

- 默认优先使用线性进度条，适合列表、表单流转和页面主流程反馈
- 空间有限时使用 `type="circle"`，更适合卡片与统计概览
- 强调目标达成率或阶段指标时使用 `type="dashboard"`
