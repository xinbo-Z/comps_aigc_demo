# SchemaForm

`SchemaForm` 是构建在基础 `Form` 之上的高阶表单组件，用于承载 schema 驱动字段渲染、动态显示规则和列表编排等能力。

## 使用示例

```tsx
import { SchemaForm } from '@sci-comp/core'

export function Example() {
  return (
    <SchemaForm
      schema={[
        {
          key: 'name',
          name: 'name',
          type: 'input',
          label: '名称',
        },
      ]}
    />
  )
}
```
