# @sci-comp/core

`@sci-comp/core` 是组件库核心包，负责提供通用组件、类型导出与主题能力。

## 你可以从这里了解什么

- 基础组件与对外导出入口
- 主题 token 的默认能力与扩展方式
- 与文档站、测试工作区的协作关系

## 快速入口

- 组件与样式源码：`src/`
- 统一导出入口：`src/index.ts`
- 主题相关实现：`src/styles/`
- 文档站主题说明：[`../../apps/sci-comp-documention/docs/guide/theme-system.md`](../../apps/sci-comp-documention/docs/guide/theme-system.md)
- 文档站快速开始：[`../../apps/sci-comp-documention/docs/guide/getting-started.md`](../../apps/sci-comp-documention/docs/guide/getting-started.md)

## 常用命令

在当前工作区目录执行：

```bash
pnpm build
pnpm dev
pnpm lint
pnpm typecheck
```
