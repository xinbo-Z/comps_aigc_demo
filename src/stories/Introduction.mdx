# sci-instrument-ui

Scientific instrument component library built with React, TypeScript, Vite, and Ant Design.

## Foundation available in Phase 1

- Shared theme tokens via `createThemeTokens()`
- Ant Design `ConfigProvider` integration for component previews
- Vitest and React Testing Library setup for component verification
- Storybook 10.3.5 configured with the React + Vite framework

## Intended library areas

- General foundational components
- Instrument-specific scientific UI components
- Shared styling and typed utility modules

## Storybook purpose

Use this workspace to document, review, and validate sci-instrument-ui components as they are added to the library.
